Using Capacitance Sensors

During the first part of Week 4, I built a circuit that involved a Capacitance Sensor. In this case, I used two metal sheets that charged up from a pin from the Metro board that provided the charge. One of the metal sheets would be connected to ground and the other to the circuit. Another pin from the Metro Board was connected to the circuit, that would sense the charging up. The close the two metal sheets were together, the higher the capacitance is for the sensor(metal sheets). Based on this correlation, it can be noted that the time it takes to fully charge up the CAP is determined by the amount of capacitance, which is determined by the distance between the two metal sheets.

The Arduino program record how long (time) it takes for the metal plates to fully charge and displays it on the application. By positioning the metal sheets in different ways, I observed many types of data changes on the Arduino Program:

When both plates are in contact with either hand: -2

When both plates touch each other completely: 20000-80000 initially; then remains at -2

When both metal sheets are in partial contact with each other: 3000-5000

I re-used the same circuit, but with a capacitance sensor that I made using home materials, of aluminum foil and a sponge. This time, instead of recording the numbers displayed on the Arduino program based on position of the two sheets, I instead came up with a new method of testing the sensor. I decided to use different writing utensils on a plate of the sensor to see how it affects the charge time. And here were the results:

Using a Pen: 1200-1800

Unsharpened Pencil: 2000

Regular sharpened Pencil: 3000

Eraser: 1800-2100

No writing object used: 10-70

It could be a possibility, that this observation method could be used to create a way to test and see the structural integrity of a writing object. Also, if the information (time numbers) reading on the Arduino program can be slowed down, that information could possibly be used to identify what series of numbers are produced when each alphabetical letter or number is written on the aluminum sheet. That could possibly used as a code language using the corresponding numbers that each letter produces when written.
